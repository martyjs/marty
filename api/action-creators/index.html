Action Creators API

Marty.createActionCreators(props)

To create some new action creators, you call Marty.createActionCreators passing in an object literal. It returns your action creators as a singleton.

var UserActionCreators = Marty.createActionCreators({
  addUser: UserActions.ADD_USER(function (name, email) {
    this.dispatch(name, email);
  })
});
Warning! Please avoid using es6 arrow syntax for your callbacks since they are bound to the current context.

displayName

An (optional) display name for the action creator. Used for richer debugging.

dispatch([...])

Dispatches an action payload. Any action handlers will be invoked with the given action handlers.

Returns Action. You can rollback an action by calling action.rollback().
